Date Object

The date object constructor will produce a date instance

a date instance is an object with properties and methods that contain information about the date.

  • built-in methods and properties

    • getDate

      Returns the day of the month (1-31) for the specified date according to local time

      example

      new Date().getDate(); returns 5;

    • getDay

      Returns the day of the week (0-6) for the specified date according to local time

      example

      new Date().getDay(); returns 6;

    • getFullYear

      Returns the year (4 digits for 4-digit years) of the specified date according to local time.

      example

      new Date().getFullYear(); returns '2019'

    • getHours

      Returns the hour (0-23) in the specified date according to local time.

      example

      new Date().getHours(); returns 10

    • setDate

      Sets the day of the month for a specified date according to local time.

      example

      new Date().setDate(15); changes the date objects getDate() value

    • setHours

      Sets the hours for a specified date according to local time.

      example

      new Date().setHours(15); changes the date objects getHours() value

    • toLocaleString

      Returns a string with a locality sensitive representation of this date.

      example

      new Date().toLocaleString(); returns '03/05/2019'
