Variable Declarations

Variables are names we give to data we want to be able to reference later.

A variable can hold all types of data ( boolean, string, number, objects, arrays, functions).

  • Keywords

    • var

      Declares a lexical scoped variable, optionally initializing it to a value.

      example

      var myVar = true;

    • const

      Declares a blocked scoped, local, read-only variable named constant.

      example

      const name = "Vinson";

    • let

      Declares a block scope local variable, optionally initializing it to a value.

      example

      let myTempValue = 0;

    • function

      Declares a lexical scoped function

      example

      function myFunction ( parameter1, parameter2 ) { console.log(parameter1, parameter2); }
