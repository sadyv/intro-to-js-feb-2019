Basic Dom

All websites are represented with the Document Object Model (DOM).

The Document Object has one 'child' element named body. It is another Object.

All elements that are visible to the user are objects nested in the body object.

  • Keywords

    • document

      This keyword is a variable that holds the entire website.

      example

      console.log(document);

    • createElement

      This is a method on the document object that can create new HTMLElements.

      example

      const myDiv = document.createElement('div');

    • appendChild

      After we create an element we can not see it unless we place it somewhere on the page.

      example

      document.body.appendChild(myDiv);

      We don't always have to append to the body.

      myDiv.body.appendChild(mySpan);

    • Listeners

      This will tell an element when an event happens run a function

      example

      document.body.addEventListener('click', function () { alert( 'you clicked me'); }
