String Object

The String global object is a constructor for strings or a sequence of characters.

There are two types of strings primitives that are made directly with quotes and strings that are made with the string object construction. (String('example'))

  • built-in methods and properties

    • length

      a property that returns how many characters a string has

      example

      'hello'.length; returns 5;

    • includes

      a method that return a boolean of whether the string contains another string

      example

      'hello'.contains('ell'); returns true;

    • indexOf

      A method that returns what index the string contains another string

      example

      'hello'.indexOf('ell'); returns 1;

    • replace

      A method that returns the string with a substring replaces with another string

      example

      'hello again'.replace('hello', 'goodbye'); returns 'goodbye again'.

    • split

      returns an array of strings that are split by a given delimiter

      example

      'hello again'.split(' '); returns ['hello','again']

    • substring

      returns a string from the string starting and ending at given indexes

      example

      'hello again'.substring(6,11); returns 'again'

    • trim

      returns a string from the string with the white space removed from each side

      example

      ' hello again '.trim(); returns 'hello again'
